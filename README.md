Generate them automatically using PlayWright and Shot Bro!

## Getting Started

### Already use Playwright?

```bash
npm install shotbro
```

Add to your gitignore:
```gitignore
.shotbro/out/
```

Get a key and add it to a `.env` file
```bash
SHOTBRO_API_KEY=keyGoesHere
```
